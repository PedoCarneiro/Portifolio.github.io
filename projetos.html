Lista de Projetos

  • Projeto 1: Sistema de Gerenciamento de Estoque (C)
    Um sistema que permite o cadastro, edição e exclusão de produtos em um estoque, com relatórios de entrada e saída, desenvolvido para ajudar pequenos comerciantes a gerenciar seus inventários de forma eficiente.

  • Projeto 2: Jogo da Forca (C++)
    Um jogo interativo da forca, onde o usuário deve adivinhar uma palavra secreta, utilizando conceitos de programação orientada a objetos para organizar o código e facilitar a expansão de funcionalidades.

  • Projeto 3: Plataforma de Aprendizado Online (C#)
    Uma aplicação web que permite aos usuários se inscreverem em cursos, assistirem a aulas em vídeo e realizarem quizzes, desenvolvida com ASP.NET e Entity Framework para a gestão de dados.

  • Projeto 4: Portfólio Pessoal (HTML/CSS)
    Um site pessoal criado para exibir meus projetos e experiências, com um design responsivo que se adapta a diferentes tamanhos de tela, utilizando boas práticas de acessibilidade e usabilidade.

  • Projeto 5: Jogo de Adivinhação de Números (JavaScript)
    Um jogo simples em que o usuário deve adivinhar um número aleatório gerado pelo computador, com feedback sobre cada tentativa, desenvolvido para praticar manipulação do DOM e eventos em JavaScript.

  • Projeto 6: Sistema de Reservas de Hotel (C#)
    Um sistema que permite a administração de reservas de hotel, onde os usuários podem verificar a disponibilidade de quartos, fazer reservas e cancelar, utilizando uma interface amigável e banco de dados SQL.

  • Projeto 7: Conversor de Moedas (Java)
    Um aplicativo que permite ao usuário converter valores entre diferentes moedas utilizando taxas de câmbio atualizadas, desenvolvido para praticar lógica de programação e manipulação de APIs.

  • Projeto 8: Jogo de Plataforma (Python)
    Um jogo de plataforma 2D desenvolvido em Python utilizando a biblioteca Pygame, onde o usuário controla um personagem que deve superar obstáculos e coletar itens, visando promover o aprendizado em desenvolvimento de jogos.
